Calculadora 3D

Uma calculadora em 3D completamente interativa feita com Three.js, os modelos 3D utilizados neste projeto foram obtidoas a partir do Poly Pizza. Sendo a calculadora uma modificação de um modelo do Bruno Oliveira. O código fonte desse projeto está dispoivel no GitHub juntamente com os créditos para os artistas responsaveis por cada um dos modelos utilizados.

Controles

  • Para ligar basta clicar no botão vermelho no lado direito
  • Mude o tamanho da fonte usando o slider em cima do teclado
  • Pressione e arraste o mouse para movimentar o modelo
  • Use a roda do mouse para dar zoom
  • Aumente o volume para uma melhor experiência
