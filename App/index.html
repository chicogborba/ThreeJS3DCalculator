Calculadora 3D

Uma calculadora em 3D completamente interativa feita com Three.js. Os modelos 3D utilizados neste projeto foram obtidos a partir do Poly Pizza. Sendo a calculadora uma modificação de um modelo de Bruno Oliveira. O código-fonte deste projeto está disponível no GitHub, juntamente com os créditos para os artistas responsáveis por cada um dos modelos utilizados.

Controles

  • Para ligar, basta clicar no botão vermelho do lado direito
  • Mude o tamanho da fonte usando o slider acima do teclado
  • Clicar no papel faz com que ele seja rasgado
  • Pressione e arraste o mouse para mover o modelo
  • Use a roda do mouse para dar zoom
  • Aumente o volume para uma melhor experiência
